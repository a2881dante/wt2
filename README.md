# Wezom Test

##Установка

- Установить composer и все зависимости

```
composer install
```

- Провести миграцию БД

```
php artisan migrate
```

- Сидировать данные в БД

```
php artisan db:seed
```

- Запустить локальный сервер


```
php artisan serve --port={port}
```
